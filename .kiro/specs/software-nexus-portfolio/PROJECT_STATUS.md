## üé¨ Feature Showcase

### Scroll-Driven Experience
Users scroll through the portfolio and camera follows a predetermined path:
```
Entry Portal ‚Üí Central Nexus ‚Üí Zones ‚Üí Projects ‚Üí Knowledge Core ‚Üí Resume
```

### Project Portals
When users click a project:
1. Camera transitions smoothly into portal
2. Project interior scene loads
3. Multiple visualization layers appear
4. User can explore architecture, flow, and story
5. Click exit to return to main world

### Interior Scene Layers
Within each project interior:
1. **Technical Layer** (Top): Architecture, data flow, metrics
2. **Decision Layer** (Middle): Why decisions were made
3. **Narrative Layer** (Bottom): Story and technology choices
